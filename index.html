Project on console

Microsoft Visual Studio is an integrated development environment from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps. First download visual studio on your device. Then open it and click on 'File' on the top left corner. Click on new file and then save your file.

The console Logs information associated with a web page: network requests, JavaScript, CSS, security errors and warnings as well as error, warning and informational messages explicitly logged by JavaScript code running in the page context. You can open the console by right clicking on the screen and clicking on 'inspect' then go to the console tab to view your console
